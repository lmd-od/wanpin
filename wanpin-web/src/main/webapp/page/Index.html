力方国际精品项目案例

  • 规划馆

    成都市武侯区城市规划馆

    通过对武侯大环境与“武侯智慧”的营造，产生一种传承的力量，让人切实感知武侯区充满智慧的蓬勃活力与触手可及的美好未来！

  • 博物馆

    西南民族大学博物馆

    西南民族大学博物馆是西南民族大学所属专业博物馆。馆藏反映了西南众多少数民族不同的历史文化和风俗民情。

  • 科技馆

    北京文化创意产业展示中心

    文创中心以时光隧道、影像长廊、动漫星球、炫幻魔方、4D影院共同表现了北京文化创意产业的高速发展，一幅绘声绘色的创意北京现代画卷徐徐展开。

  • 企业馆

    掌上明珠家具文化品牌馆

    掌上明珠家具文化品牌馆以其独有的艺术形象表现这个行业的悠久历史，展示掌上明珠的品牌、实力和人文。

  • 体验馆

    三大“难”高音全国巡演北京站

    力方国际为三位“难高音”在北京精心布展策划，舞台效果得到惊人提升，让吴宗宪、康康、高凌风三位“搞笑大哥”的幽默风暴席卷全场。

  • 军史馆

    某军区360作战指挥系统

    本项目使用多点触碰技术，将整个液晶屏幕还原成军区作战指挥部的现场。

  • 地产动画

    越秀岭南隽庭

    影片中的运用一些文化色彩浓厚的静物特写，结合画面中静物的轻盈摆动，光影的优美变换以通感的手法直观地呈现出来。

  • 城市宣传片

    2016里约奥运会主体育馆

    2016年里约热内卢奥运会将于2016年8月5日-21日在巴西的里约热内卢举行，力方团队为主体育馆提供图像视觉服务。

  • 原创动画

    傲剑

    《傲剑》是款全3d大型角色扮演类网络游戏制作的全新高清CG片头，力方国际借助团队成熟的CG技术保证了影片的顺利完成。

  • 互动多媒体

    特种影院/数字沙盘/体感互动/虚拟仿真

    多媒体技术是使用计算机交互式综合技术和数字通信网络技术处理文本、图形、图像、视频和声音，集成为一个交互系统广泛应用于各类展馆及售楼中心等领域。

  • 智慧城市

    宁夏智慧城市展示中心

    建设“智慧宁夏”，树立宁夏对外开放新形象，打造国内领先的智慧城市样本。

  • 售楼中心

    广东清远天安智谷产业园售楼中心

    天安智谷主要针对招商之用。利用智慧园区的概念，可以将企业的视角覆盖到全世界任何一个地方，经营环境自然更加优越。
